1. ~はずがない、~わけがない [ ne peut pas être (impossible) ]
あの店は今日休みのはずはありません。電話で確認したんですから。
Ce magasin ne peut pas fermer aujourd'hui. J'ai appelé pour confirmer.

2. ~とは限らない [ pas nécessairement vrai; ce n'est pas toujours vrai ]
新聞に書いてあることがいつも本当のことだとは限らない。
Ce que vous lisez dans les journaux n’est pas toujours vrai.

3. ~わけではない、~というわけではない、~のではない [ ce n'est pas…mais plutôt ]
この仕事が好きだというわけではないが、彼といっしょうに仕事が出来て楽しい。
Ce n’est pas que j’aime ce travail, mais j’aime travailler avec lui.

4. ~ないことはない [ il pourrait y avoir une possibilité que... / ce n'est pas absolument certain que ]
ここから駅まで歩けないことはありませんが、かなり時間がかかりますよ。
Il n'est pas impossible de marcher jusqu'à la gare à partir d'ici, mais cela prendra un certain temps.

5. ~ことは~が、… [  ]
彼からの手紙は読んだことは読んだですが、意味がよく分かりませんでした。
J'ai lu sa lettre, mais je n'ai pas vraiment compris ce qu'elle signifiait.
