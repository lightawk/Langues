1. ~(の)なら [ si ; dans le cas où (supposition) ~ ]
その本、読んでしまったの[なら]、私に貸してくれませんか。
Si vous avez lu ce livre, pourriez-vous me le prêter ?

2. ~ては…、~(の)では…
そんな無責任な態度[では]みんなに嫌われますよ。
Tout le monde déteste cette attitude irresponsable.

3. ~さえ~ば…、~さえ、~なら… [ si seulement ~ alors ~ ; tant que ~ ]
一言[ごめんなさい。]と言い[さえすれば]、相手は許してくれるだろう。
Si vous dites simplement « je suis désolé », l’autre personne vous pardonnera.

4. たとえ~ても…、たとえ~でも… [ même si… c'est le cas ]
たとえ面倒でも、健康診断は毎年受けたほうがいいですよ。
Même si cela peut être compliqué, c'est une bonne idée de faire un bilan de santé chaque année.

5. ~ば…、~たら…、~なら… [ si… ]
お金と暇があれば、私も、海外旅行するんだけど...。
Si j'avais l'argent et le temps, je voyagerais aussi à l'étranger...
